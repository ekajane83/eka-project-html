# 🚀 Senior Cloud Architect Resume - Ekaete

A stunning, futuristic one-page HTML resume designed for cloud architects and tech professionals. Features a cyberpunk-inspired aesthetic with dynamic animations, neon accents, and interactive elements.

## ✨ Features

- **🎨 Cyberpunk Aesthetic**: Dark theme with neon cyan, magenta, and purple color palette
- **💫 Dynamic Animations**: 
  - Floating gradient orbs in the background
  - Animated grid pattern that moves continuously
  - Glowing text effects on the title
  - Smooth hover interactions on all elements
  - Shimmer effects on stat cards
  - Slide-up animations on content load
- **📱 Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **⚡ Zero Dependencies**: Pure HTML/CSS with Google Fonts only
- **🎯 Interactive Elements**: 
  - Cards lift and glow on hover
  - Skill tags animate with gradient sweep
  - Smooth transitions throughout
- **🔤 Premium Typography**: 
  - Orbitron for futuristic headings
  - Space Mono for accents
  - JetBrains Mono for body text

## 🎯 Sections Included

1. **Hero Header** - Name, title, and company with glowing effects
2. **Stats Bar** - Key metrics (years of experience, projects, uptime)
3. **Expertise** - Core cloud architecture competencies
4. **Achievements** - Quantifiable accomplishments
5. **Certifications** - Professional credentials (AWS, Azure, GCP, K8s, etc.)
6. **Leadership** - Management and strategic contributions
7. **Technical Stack** - 18 interactive skill tags

## 🎨 Customization

### Update Personal Information

Edit these sections in the HTML:

```html
<!-- Name -->
<h1>EKAETE</h1>

<!-- Title -->
<div class="title">Senior Cloud Architect</div>

<!-- Company & Experience -->
<div class="meta-item">Abacus Group</div>
<div class="meta-item">8+ Years Experience</div>

<!-- Stats -->
<div class="stat-number">8+</div>
<div class="stat-label">Years in Cloud</div>
```

### Customize Colors

Modify the CSS variables at the top of the `<style>` section:

```css
:root {
    --primary: #00d9ff;      /* Cyan - main accent */
    --secondary: #ff006e;    /* Magenta - secondary accent */
    --accent: #8338ec;       /* Purple - tertiary accent */
    --dark: #0a0e27;         /* Background card color */
    --darker: #050714;       /* Main background */
    --light: #e0f7ff;        /* Text color */
}
```

### Add/Remove Sections

Each section is a card with this structure:

```html
<div class="card slide-up">
    <h2>Section Title</h2>
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <!-- Add more items -->
    </ul>
</div>
```

### Modify Skills

Update the skills grid:

```html
<div class="skills-grid">
    <div class="skill-tag">Your Skill</div>
    <div class="skill-tag">Another Skill</div>
    <!-- Add more skills -->
</div>
```
